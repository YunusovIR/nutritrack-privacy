Политика конфиденциальности

Последнее обновление: апрель 2026 г.

Ваша приватность

Все данные хранятся исключительно на вашем устройстве. NutriTrack не имеет серверов, не передаёт данные о вашем здоровье третьим лицам и не требует регистрации.

1. Оператор персональных данных

Оператором персональных данных является физическое лицо — разработчик приложения NutriTrack.
Контактный адрес: support.nutritrack@gmail.com

Настоящая Политика конфиденциальности разработана в соответствии с Федеральным законом Российской Федерации № 152-ФЗ «О персональных данных».

2. Какие данные собираются

Приложение собирает следующие данные, которые вы вводите самостоятельно:

  • Имя, возраст, пол (при онбординге) — хранится локально на вашем устройстве
  • Данные о здоровье: результаты лабораторных анализов, принимаемые добавки, время приёма — хранится локально
  • Настройки напоминаний и расписания — хранится локально

Не собираются: геолокация, контакты, история браузера, данные других приложений.

Сканер штрихкодов: изображение обрабатывается только в момент сканирования для поиска продукта в открытой базе данных Open Food Facts. Изображение не сохраняется и не передаётся разработчику.

3. Где хранятся данные

Все персональные данные и данные о здоровье хранятся исключительно на вашем устройстве в зашифрованном хранилище. Данные не передаются на серверы разработчика или каким-либо третьим лицам. При удалении приложения все данные удаляются автоматически.

4. Передача данных третьим лицам

4.1. Партнёрские ссылки (iHerb, Аптека.ру, Wildberries)

В разделе рекомендаций представлены партнёрские ссылки на магазины. Переход по ссылке осуществляется исключительно по вашему желанию. Данные о вашем здоровье, анализах или принимаемых добавках не передаются партнёрам. При переходе по ссылке магазин может получить стандартную техническую информацию о переходе (реферальный код).

4.2. Firebase Crashlytics (Google)

Для обеспечения стабильности приложения используется сервис Firebase Crashlytics. При возникновении ошибки (краша) передаются технические данные: тип устройства, версия ОС, стек вызовов в момент ошибки. Персональные данные, данные о здоровье и содержимое анализов в crash-отчётах не фигурируют.

Политика конфиденциальности Google: policies.google.com/privacy

5. Права субъекта персональных данных (152-ФЗ, ст. 14–17)

В соответствии с законодательством РФ вы имеете право:

  • Удаление данных — в Настройках приложения раздел «Опасная зона» → «Сбросить все данные». Все данные будут удалены безвозвратно.
  • Получение копии данных — экспорт данных доступен в разделе Аналитика → PDF-отчёт для врача.
  • Отзыв согласия — вы вправе отозвать согласие на обработку данных, удалив приложение. Все данные будут удалены вместе с приложением.
  • Обращение к оператору — по любым вопросам, связанным с обработкой персональных данных, направляйте запросы на электронную почту, указанную в разделе 1.

6. Дети

Приложение NutriTrack не предназначено для лиц младше 18 лет. Если вам стало известно, что ребёнок предоставил нам персональные данные без согласия родителей, пожалуйста, свяжитесь с нами по контактному адресу.

7. Медицинский дисклеймер

NutriTrack не является медицинским приложением и не предоставляет медицинских консультаций. Рекомендации по добавкам носят исключительно информационный характер. Перед началом приёма любых добавок проконсультируйтесь с врачом. БАД не являются лекарственными средствами.

8. Изменения политики

Мы можем обновлять настоящую Политику конфиденциальности. Актуальная версия всегда доступна по данному адресу. Дата последнего обновления указана в начале документа.
